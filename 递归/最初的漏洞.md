```cpp
#include <cstdio>
const int maxn = 11;
int n, P[maxn], hashTable[maxn] = {false};
void generateP[int index] {
    if (index == n + 1) {
        for (int i = 1; i <=n; i++) {
            printf("%d", P[i]);
        }
        printf("\n");
        return;
    }
    for (int x = 1; x <= n; x++) {
        if (hashTable[x] == false) {
            P[index] = x;
            hashTable[x] = true;
            generateP(index + 1);
            hashTable[x] = false;
        }
    }
}

int main() {
    n = 3;
    generateP(1);
    return 0;
}
```

这段代码一开始没理解，想了好久，最后通过查阅资料明白了递归的一个重要特性——==**每一次递归调用都会初始化函数中的所有变量**。==
